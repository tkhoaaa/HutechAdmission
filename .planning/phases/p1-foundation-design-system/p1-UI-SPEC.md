| Token | Value | Usage |

| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Default element padding, form field gaps |
| lg | 24px | Section padding, card internal padding |
| xl | 32px | Layout gaps, major element spacing |
| 2xl | 48px | Major section breaks |

**Exceptions:**
- Touch targets: minimum 44px (mobile nav items, icon buttons)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f8fafc` / oklch light | Page backgrounds |
| Secondary (30%) | `#ffffff` / oklch white | Cards, surfaces, modals |
| Accent (10%) | `#3b82f6` (blue-500) | Primary CTAs, active nav, focus rings |
| Destructive | `#ef4444` (red-500) | Delete actions, error states |

**shadcn CSS Variables (light mode):**
- `--primary: oklch(0.205 0 0)` → override to blue-500 `#3b82f6`
- `--primary-foreground: oklch(0.985 0 0)` (white)
- `--destructive: oklch(0.577 0.245 27.325)` → red-500
- `--border: oklch(0.922 0 0)` → gray-200
- `--background: oklch(1 0 0)` → white
- `--foreground: oklch(0.145 0 0)` → gray-900

**Accent reserved for (never "all interactive elements"):**
1. Primary CTA buttons (submit, register, apply)
2. Active navigation item indicator
3. Focus ring on keyboard navigation (`--ring`)
4. Primary link color on hover
5. Loading spinner color

**60/30/10 split:**
- 60% dominant: Light blue-gray backgrounds (`#f8fafc`) for page body
- 30% secondary: White cards and surfaces
- 10% accent: Blue-500 primary CTAs and interactive highlights

**Destructive color:** `#ef4444` — reserved for delete, reject, cancel enrollment actions only.

## Animation Principles

| Type | Duration | Easing |
|------|----------|--------|
| Micro-interactions | 150ms | ease-out |
| Component transitions | 200ms | ease-in-out |
| Page transitions | 300ms | ease-out |
| Entrance stagger | 50ms delay per item | — |
| Modal open | 200ms spring | stiffness: 500, damping: 30 |

**Rules:**
- Max stagger delay per section: 300ms total
- Entrance animations: fade + translateY(8px) only (no scale, no bounce)
- Hover: scale(1.02) + translateY(-2px) max
- Tap: scale(0.97) max
- Remove ALL `Math.random()` from JSX render
- Infinite animation loops: only for loading spinners, not decorative elements
- `prefers-reduced-motion`: respect via `@media (prefers-reduced-motion: reduce)`

## Accessibility

| Rule | Implementation |
|------|----------------|
| Focus visible | `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2` |
| Keyboard nav | All interactive elements reachable via Tab/Shift+Tab |
| Skip link | Hidden "Skip to main content" link at page top |
| Color contrast | All text meets WCAG AA (4.5:1 body, 3:1 large text) |
| Screen reader | Semantic HTML, ARIA labels on icon-only buttons |
| Reduced motion | `@media (prefers-reduced-motion: reduce)` disables all non-essential animations |
| Form labels | All inputs have visible labels or aria-label |
| Error messages | Associated via `aria-describedby` |
